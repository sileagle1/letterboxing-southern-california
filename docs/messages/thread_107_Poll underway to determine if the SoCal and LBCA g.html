Hmm.... Definitely need to clear up any confusion for the vote to
count! =)

Keep in mind the question is "What should be done with this talk
list?", as in the talk list where the poll was created. So the first
option is a vote for creating one, single California talk list that
encompasses the entire state. This could mean destroying both of the
existing boards and started a new group from scratch, or deleting
either of the existing boards and moving everyone over to the other
one. Those details would be worked out after this vote is done,
though.

The second option is a vote for my changing the board I created into
a Northern California board only (or rather, any area not covered by
the Southern California board--including Central California). The
Southern California board would continue as it always has with no
changes at all. (Probably what most people on this board prefer--
especially if they don't get out of Southern California much.)

There's no right or wrong answer--just personal preferences and in
the quest for making as many people possible happy with the decision,
I thought I'd put it up for a vote. No decision will make everyone
happy, unfortnately. In fact, even doing NOTHING will make some
people unhappy. It's the worst case of a lose-lose-lose situation
I've ever been a part of. But at least the stakes aren't very high.
=) Oh, if only I created a Northern Letterboxing board from day 1
and none of this conflict would have started in the first place.
