Search for boxes function

From: gwendontoo (foxsecurity@earthlink.net) | Date: 2005-04-25 22:43:40 UTC

I just checked out the new improved search for boxes function. The old
function seemed to require that the entire box name be used to find
the letterbox listing you were looking for. Now it seems to work more
like google, and if you type in one word from the title it will pull
up a listing of letterboxes with that word in the title. If you type
in "missing" for example, the listing will include all of those boxes
where the placer has put that word in the title after they have edited
the box clues.(I just thought of a great title for a letterbox-
"Missing") You no longer have to search the map list for clues in a
specific area. Instead you can type in the county name only with no
other information and it will give you a list of the boxes in that
county only. Before you needed a lot more information typed in before
you could find clues, and it was easier then to just go to the map
listings.
