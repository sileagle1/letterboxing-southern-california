For those that dislike lengthy status reports on this list, please
excuse the following post. Also this post has been cross posted
purposely as many folks do not read the main chat list.
Honor Liberty Hitch Hiker was launched 9/11/2002 and it is a Hitch
Hiker with a different purpose and different method of notification.
The finder is supposed to post on a chat list the exact location
where it is left. For background see message # 34174 and message #
46814 on the main LbNA chatlist. Those messages were listings of
comments from the letterboxers that found her. It makes pretty good
reading on this date.

We were sent the text from the logbook from the last finders, The
Foxes in Montana, and it too makes good reading.
