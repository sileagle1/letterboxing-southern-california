From: troublew_tracy (troublew_tracy@yahoo.com) | Date: 2007-03-02 20:12:51 UTC
The voting time is over, and the winning California Letterboxing Patch
is... *drumroll*... Number Nine!

As I mentioned earlier on the SoCal board, I really liked the #7
entry's idea of using "Eureka!" and I think it fits well with this
patch design, so when it began to look like this one would win, I did
some tweaking with the original image to add that in, and you can see
it here: http://i21.photobucket.com/albums/b279/tdscrafts/patch.jpg
If anyone has any objections to using this design instead of the
original, please let me know.

I would also appreciate it if people would e-mail me with how many
patches they plan on ordering, so that I can get an idea of how many to
order. I'm not sure of what the pricing will be at this point--
probably around $2-3 per patch-- but I will work on getting an estimate
soon.

Tdyans
