Carving Tips at SD Gathering

From: gwendontoo (foxsecurity@earthlink.net) | Date: 2007-03-06 04:15:48 UTC
If anyone wants to spend some time at the picnic tables going over some
carving tips please bring some tools and material with you. For those
that want to create a #1/2 speedball blade bring a few #1 blades, a
candle, matches and a pair of pliers or better yet a vise grip.
If you have never used a 1/2 you are going to love it.
