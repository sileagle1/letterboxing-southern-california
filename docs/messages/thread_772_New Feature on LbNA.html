Yes but we have long ago forsaken planting in NPS areas, and in other parks we ask for forgiveness rather than ask for permission. This does make clue writing interesting and even though we no longer plant in NPS areas, we do have some that have been in place a long,long time.

I'm not advocating planting in areas that wish permits or no plants as unless the clue sheet is properly masked it does create a problem for host web sites.

If clues are set up and called "Half Dome", "Yellowstone", or some other name that is obvious then those will get confiscated. We do have one called "Yosemite" but that is more of a tweek for NPS and it is not on NPS turf.

Fortunately around Kernville, most of the land is controlled by BLM or Forest Service and there biggest problem is illegal cash crops.
