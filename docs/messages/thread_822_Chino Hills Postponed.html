Chino Hills Postponed

From: Kelsung (kelsung42@yahoo.com) | Date: 2010-03-10 03:26:10 UTC
As some of you know, the weather has been against us. I've had high hopes that things would be okay by the time of the event. But I now have it on a strong authority (a park ranger) that Chino Hills State Park will most likely be closed this weekend. Therefore, I think we need to postpone the gathering for the following Saturday, the 20th.

They had a bulldozer level roads a few weeks ago and opened one weekend, but it apparently was decided that they want to make sure all of them are clear before opening again. Although they may get that done this week, the goal is to have it done for next week. The weather forecast has no more rain in it, so everything will be fine for the 20th.

I realize this is pretty last minute, but they were hoping as much as me. The rangers don't like having the park closed anymore than the rest of us. I know some of you may not be able to make it on the 20th instead, and I'm truly sorry. But I don't want to have 40+ folks showing up to a locked gate this weekend.

My bigger worry is that not everyone planning to come will get the message, so please help spread the news. I think we can still have a successful event on the 20th.

Kelsung
